# RazorConsole v0.2.2

**Released: December 5, 2025** | [Milestone v0.2.2](https://github.com/RazorConsole/RazorConsole/milestone/3)

RazorConsole v0.2.2 introduces a new StepChart component for data visualization, improves the documentation website with better navigation and performance, and delivers major codebase refactoring for improved maintainability and reliability.

## Highlights

- **New StepChart Component**: Added `StepChart` component for step/line graph visualization with customizable series, colors, and axis labeling ([#219](https://github.com/RazorConsole/RazorConsole/pull/219))
- **Repository Migration**: Moved RazorConsole to the new [RazorConsole organization](https://github.com/RazorConsole) on GitHub ([#229](https://github.com/RazorConsole/RazorConsole/pull/229))
- **Major Codebase Refactoring**: Improved ConsoleRenderer architecture, added thread safety with `System.Threading.Lock`, migrated tests to Shouldly assertions, and added code coverage reporting ([#221](https://github.com/RazorConsole/RazorConsole/pull/221))
- **Website Navigation Improvements**: Added hash-based documentation navigation, sticky sidebar, and markdown rendering optimization ([#233](https://github.com/RazorConsole/RazorConsole/pull/233))
- **Focus Management Documentation**: Comprehensive docs for controlling focus behavior in console applications ([#212](https://github.com/RazorConsole/RazorConsole/pull/212))
- **New Collaborators Page**: Added a dedicated page showcasing project contributors with bios and social links ([#231](https://github.com/RazorConsole/RazorConsole/pull/231))

## Detailed Changes

### Components & Rendering

- **New Components**:
  - Added `StepChart` component with series support, customizable colors, axis labeling, and title styling ([#219](https://github.com/RazorConsole/RazorConsole/pull/219))
  - Added StepChart gallery example and website preview ([#219](https://github.com/RazorConsole/RazorConsole/pull/219))

- **Rendering Improvements**:
  - Fixed render region updates for more reliable display output ([#239](https://github.com/RazorConsole/RazorConsole/pull/239))

### Core Refactoring

- Refactored `ConsoleRenderer` for improved readability and maintainability ([#221](https://github.com/RazorConsole/RazorConsole/pull/221))
- Added code-generated logger extensions for proper logging support ([#221](https://github.com/RazorConsole/RazorConsole/pull/221))
- Improved thread safety using `System.Threading.Lock` and fixed code duplication ([#221](https://github.com/RazorConsole/RazorConsole/pull/221))
- Migrated test assertions to Shouldly for improved readability ([#221](https://github.com/RazorConsole/RazorConsole/pull/221))
- Added code coverage reporting integration ([#221](https://github.com/RazorConsole/RazorConsole/pull/221))
- Migrated to mirror test structure for easier navigation ([#221](https://github.com/RazorConsole/RazorConsole/pull/221))

### Website & Documentation

- **Documentation Improvements**:
  - Added focus management documentation ([#212](https://github.com/RazorConsole/RazorConsole/pull/212), fixes [#129](https://github.com/RazorConsole/RazorConsole/issues/129))
  - Added PlantUML diagrams to design-docs showing rendering pipeline ([#214](https://github.com/RazorConsole/RazorConsole/pull/214), fixes [#208](https://github.com/RazorConsole/RazorConsole/issues/208))
  - Fixed and updated documentation links, headers, and style issues across website, design-docs, and README ([#236](https://github.com/RazorConsole/RazorConsole/pull/236))

- **Website Enhancements**:
  - Added hash-based routing for documentation sections with automatic scrolling ([#233](https://github.com/RazorConsole/RazorConsole/pull/233))
  - Optimized markdown component with React.memo to prevent unnecessary re-renders ([#233](https://github.com/RazorConsole/RazorConsole/pull/233))
  - Fixed CSS for sticky sidebar navigation ([#233](https://github.com/RazorConsole/RazorConsole/pull/233))
  - Added WASM lazy-loading and LoadingOverlay component for improved initial load performance ([#238](https://github.com/RazorConsole/RazorConsole/pull/238))
  - Added collaborators page with contributor bios, avatars, roles, and social media links ([#231](https://github.com/RazorConsole/RazorConsole/pull/231))
  - Updated footer documentation links ([#237](https://github.com/RazorConsole/RazorConsole/pull/237))

### Build, Packaging & Repository

- Updated all repository URLs from `LittleLittleCloud/RazorConsole` to `RazorConsole/RazorConsole` ([#229](https://github.com/RazorConsole/RazorConsole/pull/229))
- Added NuGet package icon ([#211](https://github.com/RazorConsole/RazorConsole/pull/211), [#213](https://github.com/RazorConsole/RazorConsole/pull/213))
