# RazorConsole v0.2.0

**Released: November 26, 2025** | [Milestone v0.2.0](https://github.com/LittleLittleCloud/RazorConsole/milestone/1)

RazorConsole v0.2.0 brings significant enhancements to the component library, improves developer experience with routing support and code style automation, and delivers numerous quality-of-life improvements across documentation, testing, and tooling.

Experience RazorConsole components in action with our new real-time preview feature on the [documentation website](https://littlelittlecloud.github.io/RazorConsole/components)â€”try components directly in your browser without any setup!

## Highlights

- **New Chart Components**: Added `BarChart` and `BreakdownChart` components with rich customization options ([#160](https://github.com/LittleLittleCloud/RazorConsole/pull/160), [#164](https://github.com/LittleLittleCloud/RazorConsole/pull/164))
- **Blazor Routing Support**: Full routing integration for navigation and multi-page console applications ([#123](https://github.com/LittleLittleCloud/RazorConsole/pull/123), [#132](https://github.com/LittleLittleCloud/RazorConsole/pull/132))
- **Enhanced Table Component**: Added border color customization ([#147](https://github.com/LittleLittleCloud/RazorConsole/pull/147))
- **Real-Time Component Preview on RazorConsole Website**: Interactive component previews now run directly in the browser, enabling live testing and development ([#163](https://github.com/LittleLittleCloud/RazorConsole/pull/163), [#199](https://github.com/LittleLittleCloud/RazorConsole/pull/199), [#191](https://github.com/LittleLittleCloud/RazorConsole/pull/191))
- **Select Component Improvements**: New `FocusedValue` binding enables dynamic UI updates during navigation ([#148](https://github.com/LittleLittleCloud/RazorConsole/pull/148))
- **Modern Solution Format**: Migration to `.slnx` format for cleaner project management ([#167](https://github.com/LittleLittleCloud/RazorConsole/pull/167))
- **Automated Code Style**: Comprehensive `.editorconfig` rules with enforced formatting ([#153](https://github.com/LittleLittleCloud/RazorConsole/pull/153))
- **Modern Testing Platform**: Migrated to Microsoft Testing Platform (MTP) with xUnit v3 for faster, native test execution ([#179](https://github.com/LittleLittleCloud/RazorConsole/pull/179))
- **Improved Documentation**: Enhanced syntax highlighting, new routing documentation, and better contribution guides ([#150](https://github.com/LittleLittleCloud/RazorConsole/pull/150), [#137](https://github.com/LittleLittleCloud/RazorConsole/pull/137))

## Detailed Changes

### Components & Rendering

- **New Components**:
  - Added `BarChart` component with label styling, value display, and culture support ([#160](https://github.com/LittleLittleCloud/RazorConsole/pull/160))
  - Added `BreakdownChart` component for proportional data visualization ([#164](https://github.com/LittleLittleCloud/RazorConsole/pull/164))
  - Created unified Charts gallery page combining both chart components ([#164](https://github.com/LittleLittleCloud/RazorConsole/pull/164))

- **Component Enhancements**:
  - Added `FocusedValue` bindable parameter to `Select` component for tracking highlighted options ([#148](https://github.com/LittleLittleCloud/RazorConsole/pull/148))
  - Added `BorderColor` property to `Table` component ([#147](https://github.com/LittleLittleCloud/RazorConsole/pull/147))
  - Fixed `Panel` component border and title color rendering with hex values ([#165](https://github.com/LittleLittleCloud/RazorConsole/pull/165))
  - Added XML documentation to all RazorConsole.Core components ([#200](https://github.com/LittleLittleCloud/RazorConsole/pull/200))

### Real-Time Preview Components in Browser for Docs Website

- **Interactive Previews**:
  - Implemented WASM rendering of Razor components in browser using xterm.js ([#163](https://github.com/LittleLittleCloud/RazorConsole/pull/163))
  - Added dynamic terminal resizing with automatic re-rendering ([#199](https://github.com/LittleLittleCloud/RazorConsole/pull/199))
  - Implemented comprehensive keyboard support (Ctrl keys, function keys, navigation keys) ([#191](https://github.com/LittleLittleCloud/RazorConsole/pull/191))
  - Added clipboard support for copy/paste operations ([#191](https://github.com/LittleLittleCloud/RazorConsole/pull/191))

### Routing & Navigation

- Added Blazor routing support for multi-page console applications ([#123](https://github.com/LittleLittleCloud/RazorConsole/pull/123))
- Created routing documentation and examples ([#132](https://github.com/LittleLittleCloud/RazorConsole/pull/132))
- Refactored gallery app to utilize `MainLayout.razor` with routing ([#139](https://github.com/LittleLittleCloud/RazorConsole/pull/139))

### Tooling, Documentation & Website

  - Created issue templates for bug reports and feature requests ([#137](https://github.com/LittleLittleCloud/RazorConsole/pull/137))

- **Website Enhancements**:

  - Fixed style bugs including overflow issues and terminal theme switching ([#183](https://github.com/LittleLittleCloud/RazorConsole/pull/183))
