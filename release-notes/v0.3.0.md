# RazorConsole v0.3.0

**Released: December 26, 2025** | [Milestone v0.3.0](https://github.com/RazorConsole/RazorConsole/milestone/4)

RazorConsole v0.3.0 introduces Native AOT support, a new Scrollbar component for better navigation, Razor syntax highlighting, auto-generated API documentation with DocFX, and delivers critical performance improvements for text input handling.

## Highlights

- **Native AOT Support**: Added Ahead-Of-Time compilation support with proper reflection annotations and trimming protection ([#263](https://github.com/RazorConsole/RazorConsole/pull/263), fixes [#262](https://github.com/RazorConsole/RazorConsole/issues/262))
- **Extend Scrollable Component**: Added customizable `Scrollbar` parameter in `Scrollable` component with embedded rendering support for Tables and Panels, providing visual feedback for scrollable content ([#256](https://github.com/RazorConsole/RazorConsole/pull/256), fixes [#255](https://github.com/RazorConsole/RazorConsole/issues/255))
- **Razor Syntax Highlighting**: Added native Razor language syntax highlighting support in code components with enhanced gallery examples ([#264](https://github.com/RazorConsole/RazorConsole/pull/264))
- **Auto-Generated API Documentation**: Integrated DocFX to automatically generate component documentation from XML comments, eliminating redundant manual documentation ([#258](https://github.com/RazorConsole/RazorConsole/pull/258), [#259](https://github.com/RazorConsole/RazorConsole/pull/259), fixes [#248](https://github.com/RazorConsole/RazorConsole/issues/248), [#257](https://github.com/RazorConsole/RazorConsole/issues/257))
- **TextInput Performance Fix**: Resolved critical freeze issue when pasting large text chunks by implementing input batching, reducing 500+ render cycles to a single cycle ([#251](https://github.com/RazorConsole/RazorConsole/pull/251), fixes [#250](https://github.com/RazorConsole/RazorConsole/issues/250))
- **Middleware Architecture**: Migrated to a cleaner middleware approach for rendering pipeline ([#249](https://github.com/RazorConsole/RazorConsole/pull/249))

## Detailed Changes

### Native AOT & Performance

- **AOT Support**:
  - Added `DynamicallyAccessedMembers` attributes to `AppHost` for AOT compiler metadata preservation ([#263](https://github.com/RazorConsole/RazorConsole/pull/263))
  - Refactored `ComponentMarkupUtilities.ResolveSpinner` with safe reflection using `DynamicallyAccessedMembers` ([#263](https://github.com/RazorConsole/RazorConsole/pull/263))
  - Fixed Blazor Router trimming issues with `TrimmerRootAssembly` protection ([#263](https://github.com/RazorConsole/RazorConsole/pull/263))
  - Added custom `.targets` file (`RazorConsole.Core.targets`) for automatic anti-trimming protection in NuGet packages ([#263](https://github.com/RazorConsole/RazorConsole/pull/263))
  - Added GitHub alerts support in Markdown component ([#263](https://github.com/RazorConsole/RazorConsole/pull/263))

### Components & Rendering

- **New Components**:
  - Extended `Scrollable` component with scrollbar. It has full customization, focus support, and embedded rendering for Tables and Panels/Borders ([#256](https://github.com/RazorConsole/RazorConsole/pull/256))
  - Added Razor syntax highlighting support with extended gallery examples ([#264](https://github.com/RazorConsole/RazorConsole/pull/264))

- **Component Improvements**:
  - Removed unnecessary nullable string in Select component's ValueChange callback to fix ReSharper variance warnings ([#261](https://github.com/RazorConsole/RazorConsole/pull/261))
  - Changed default TextButton FocusedColor from Yellow to DeepSkyBlue1 for better visual appearance ([#269](https://github.com/RazorConsole/RazorConsole/pull/269))

### Focus & Input Reliability

- **Performance Improvements**:
  - Fixed TextInput freeze when pasting large text (500+ characters) by batching consecutive key inputs ([#251](https://github.com/RazorConsole/RazorConsole/pull/251))
  - Implemented input batching in KeyboardEventManager with 1000 character safety limit ([#251](https://github.com/RazorConsole/RazorConsole/pull/251))
  - Added three new tests for sequential character handling and backspace behavior ([#251](https://github.com/RazorConsole/RazorConsole/pull/251))

### Core Architecture

- Migrated to middleware approach for improved rendering pipeline ([#249](https://github.com/RazorConsole/RazorConsole/pull/249))

### Website & Documentation

- **Auto-Generated Documentation**:
  - Integrated DocFX to generate API documentation from XML comments ([#258](https://github.com/RazorConsole/RazorConsole/pull/258))
  - Fixed GitHub Pages documentation rendering ([#259](https://github.com/RazorConsole/RazorConsole/pull/259))
  - Added missing components to components.generated.ts ([#260](https://github.com/RazorConsole/RazorConsole/pull/260))

- **Website Enhancements**:
  - Fixed xterm color rendering to support origin colors in syntax highlighting ([#265](https://github.com/RazorConsole/RazorConsole/pull/265))
  - Extended Select component documentation with missing properties ([#245](https://github.com/RazorConsole/RazorConsole/pull/245))
  - Fixed lineHeight from 1.2 to 1 in XTermPreview.tsx for correct component rendering ([#270](https://github.com/RazorConsole/RazorConsole/pull/270))

### Build & Versioning

- Added v0.2.2 release notes to website and bumped version to 0.3.0 ([#247](https://github.com/RazorConsole/RazorConsole/pull/247))
